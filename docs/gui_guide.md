# GUI Guide

## Overview
This guide walks you through the entire web interface of the Advanced Trading Bot Platform, covering all screens, user flows, and advanced features.

---

## 1. Dashboard Overview
- **Widgets:** Total P&L, Active Strategies, Open Positions, Win Rate
- **Charts:** Portfolio performance, asset allocation
- **Recent Trades:** Table of latest trades
- **Strategy Status:** Cards for each running strategy
- **Balance Protection:** Daily loss, drawdown, protected balance
- **AI Recommendations:** LLM/AI suggestions for improvement
- **Failsafe Panel:** Pause all, emergency stop

---

## 2. Profile & Account Management
- **Profile:** View/edit user info, change password, 2FA
- **Accounts:** Add/remove trading accounts, set default
- **Wallets:** Connect/disconnect crypto wallets, MetaTrader accounts
- **API Keys:** Manage API keys for exchanges, brokers

---

## 3. Wallet & MetaTrader Linking
- **Add Wallet:** Click “Connect Wallet”, choose type (crypto/MetaTrader), follow prompts
- **View Balances:** See all connected wallets/accounts and balances
- **Withdraw:** Initiate withdrawal, set % deduction for compounding
- **Security:** Set withdrawal whitelist, enable withdrawal protection

---

## 4. Strategy Creator & Management
- **Create Strategy:** Use the wizard or upload a trading plan (JSON/CSV)
- **Edit Strategy:** Change parameters, risk, indicators, allocations
- **Activate/Pause/Stop:** Start, pause, or stop strategies
- **Allocation:** Assign % of balance to each strategy
- **Failsafe:** Set max loss, drawdown, emergency stop triggers
- **AI Suggest:** Get LLM/AI recommendations for settings
- **Upload Plan:** Import your own trading plan for review and optimization

---

## 5. Backtester & Analytics
- **Run Backtest:** Select strategy, asset, date range, and run
- **Results:** View equity curve, drawdown, monthly returns, trade log
- **Optimization:** Use AI/ML to optimize parameters
- **Compare:** Compare multiple strategies side-by-side
- **Export:** Download results as CSV or PDF

---

## 6. Live Trade Viewer & Management
- **Live Chart:** Real-time price chart (TradingView)
- **Quick Trade:** Place market/limit/stop orders
- **Open Positions:** View, modify, or close positions
- **Pending Orders:** View/cancel/modify orders
- **Recent Trades:** Table of executed trades
- **Emergency Close:** One-click close all positions

---

## 7. Balance & Allocation Tracking
- **Balances:** See all account and wallet balances
- **Allocation:** Visualize allocation per strategy/account
- **Compounding:** Enable/disable compounding, set withdrawal %
- **Protection:** Configure daily loss, drawdown, and risk settings

---

## 8. Failsafe & Emergency Controls
- **Pause All:** Pause all strategies instantly
- **Emergency Stop:** Kill all strategies and close all positions
- **Alerts:** View and acknowledge risk/failsafe alerts
- **History:** See log of all emergency actions

---

## 9. AI/ML Recommendations
- **AI Suggest:** Get LLM/AI suggestions for any strategy or portfolio
- **Review:** Let AI review performance and suggest tweaks
- **Parameter Tuning:** Accept or edit AI-recommended settings
- **Full Access:** AI can analyze all strategies, trades, and settings

---

## 10. Customization & Settings
- **Theme:** Light/dark mode
- **Notifications:** Email, SMS, in-app alerts
- **Language:** Multi-language support
- **API Access:** Generate/manage API keys
- **Security:** 2FA, session management, device whitelist

---

## 11. FAQ & Troubleshooting
- **Strategy not performing?** Use AI Review for diagnostics
- **Can’t connect wallet?** Check API keys and permissions
- **Emergency stop not working?** Check network and backend status
- **Need more help?** Use the in-app support or see [API Guide](api_guide.md)
